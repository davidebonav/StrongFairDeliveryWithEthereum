## Installazione 
Per installare le dipendenze con npm
```shell
    npm install
```
Per eseguire i test dello Smart Contract con Hardhat
```shell
    npx hardhat test
```

Per ottenere la _code coverage_ dello Smart Contract
```shell
    npx hardhat coverage
```
